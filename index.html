Numix

Telegram bot for some numerical operations that combine math with text.

Math

Mathematical and general percentage operations like adding or subtracting percent of value ($20 + 40%), you also can use additional percentage operations.

  • Example Operation
  • 1.2 / (3.3 + 1.7) Direct mathematical operations
  • 5% on $30 Adding percentage
  • 6% off €40 Substracting percentage
  • 20% of $10 Percentage value
  • 5% of what is €6 Value by percent part
  • $50 as a % of $100 Percentage value of one value relative to another

Currency

Current and historical foreign exchange (forex) rates published by the European Central Bank. Rates update daily around 4PM CET every working day. Historical rates go back to 1st January, 1999.

You can get foreign exchange reference rates and convert it. Use ISO 4217 codes for currency (like USD, RUB, CAD etc).

  • Example Operation
  • /cc Currency selection on the keyboard
  • /cc usd Get the latest foreign exchange reference rates
  • /cc usd for 2017-12-17 Get historical rates for any day since 1999
  • /cc 17.18 usd to rub Currency conversion
  • /cc 17 usd to rub for 2013-12-10 Currency conversion with historical rates

Timezone

Setting the timezone will be necessary if you use the note commands. You can set and change it.

  • Example Operation
  • /tz Get timezone
  • /tz Moscow Set your timezone

Notes

You can use note commands to create some text notifications, also you can get currency exchange rates in daily notifications (look examples).

  • Example Operation
  • /note 12:20 - Eat! Set note
  • /note 12:20 - /cc usd Set currency commands in note
  • /note_ls Get note list
  • /note_rm 1 Remove note
  • /note_rm all Remove all notes

Another

Some additional operations, like time.

  • Example Operation
  • /time Moscow Get time for city
